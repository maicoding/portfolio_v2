CLAUDIA MAI / PORTFOLIO V2

Manifest Works Method

CH_00 // SIGNAL

A portfolio that behaves like a live transmission.

CH_01 // MANIFEST

01 / Haltung

Gestaltung ist hier nicht nur Oberflaeche, sondern Positionierung. Jedes Projekt hat einen Konflikt, eine Methode und eine Konsequenz.

02 / Material

Bilder, Daten, Textfragmente, Zufall, KI, Handarbeit: Alles wird als Material verstanden, nicht als fertige Wahrheit.

03 / Erzaehlung

Statt linearem "Projekt aufrufen" entsteht ein Storyflow. Besucherinnen bewegen sich kapitelweise durch Positionen, Brueche und Ergebnisse.

CH_02 // WORK INDEX

Kachel waehlen, um die Story-Ebene zu oeffnen.

CH_03 // METHOD

  1. Observe

    Recherche beginnt im Kontext: Bildpolitik, Plattformlogik, soziale Situation.

  2. Distort

    Material wird verschoben: durch KI, Uebersetzung, Re-Layout, Hybridisierung.

  3. Compose

    Formate entstehen als Systeme: Print, Raum, Web, Performance, AR.

  4. Reflect

    Jedes Ergebnis wird als Aussage getestet: Was zeigt es? Was verschweigt es?
